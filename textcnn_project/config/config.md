# TextCNN 模型参数汇总

| 参数 | 当前设置 | 说明 |
|------|----------|------|
| **embedding 维度 (`embed_dim`)** | 200 | 每个单词的向量维度，用于表示语义特征 |
| **卷积通道数 (`num_channels`)** | 200 | 每个卷积核输出的通道数，通道越多，模型表达能力越强 |
| **上下文窗口大小 / 卷积核 (`kernel_sizes`)** | `[2,3,4,5]` | 文本卷积核大小，对应 n-gram 特征长度：2 → 2-gram，3 → 3-gram，4 → 4-gram，5 → 5-gram |
| **dropout** | 0.5 | 防止过拟合，训练时随机丢弃部分神经元 |
| **类别数 (`num_classes`)** | 2 | 1 → 负面，2 → 正面 |
| **最大序列长度 (`MAX_LEN`)** | 200 | 文本序列截断或填充到固定长度 |
| **批量大小 (`BATCH_SIZE`)** | 32 | 每次训练处理的样本数量 |
| **训练轮数 (`EPOCHS`)** | 10 | 总训练轮次 |
| **学习率 (`LR`)** | 1e-3 | Adam 优化器学习率 |
| **损失函数 (`criterion`)** | `CrossEntropyLoss(weight=class_weights)` | 根据训练集标签平衡类别，防止偏向多数类 |
| **设备 (`DEVICE`)** | 自动选择 GPU/CPU | 有 GPU 则使用 GPU，否则使用 CPU |
