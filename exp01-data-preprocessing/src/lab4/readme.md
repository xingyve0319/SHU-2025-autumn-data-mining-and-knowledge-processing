T-SNE 可视化脚本使用说明 (readme)

本脚本用于对 Word2Vec 与 Node2Vec 训练好的向量进行 t-SNE降维与可视化。

------------------------------------------------------------------------

一、基本运行方式

    python T-SNE.py [参数]

脚本将自动： - 加载预训练的 Word2Vec / Node2Vec 模型
- 对向量执行 t-SNE
- 输出到 Part4/ 目录

------------------------------------------------------------------------

二、命令行参数说明

1. 数据规模设置

  参数               默认值   说明
  ------------------ -------- -----------------------------------
  --word-size, -ws   200      可视化的词向量数量
  --node-size, -ns   300      可视化的节点数量（针对 Node2Vec）

------------------------------------------------------------------------

2. 可视化设置

  参数             默认值   说明
  ---------------- -------- ---------------------------
  --annotate, -a   20       图中标注的词或节点数量
  --figsize, -f    15 12    图片尺寸（宽 高）
  --method, -m     tsne     降维方法（已固定为 tsne）

------------------------------------------------------------------------

3. 控制开关

  参数                说明
  ------------------- ------------------------------
  --show-plots, -s    显示图像（默认只保存不显示）
  --skip-word2vec     不可视化 Word2Vec
  --skip-node2vec     不可视化 Node2Vec
  --skip-comparison   不执行 Word2Vec 模型对比
  --skip-semantic     不执行语义聚类分析

------------------------------------------------------------------------

三、运行示例

1. 最基础运行（默认 t-SNE）

    python T-SNE.py

2. 指定词数量并展示图像

    python T-SNE.py -ws 300 -s

3. 自定义图像尺寸

    python T-SNE.py -f 18 14

4. 仅运行 Node2Vec 可视化

    python T-SNE.py --skip-word2vec

------------------------------------------------------------------------

四、输出结果

所有结果保存到：

    Part4/

     └── visualization_analysis_report.txt

------------------------------------------------------------------------

五、注意事项

-   只使用 t-SNE 进行降维
-   需确保模型存在：

    -   Part1/word2vec_sg.model
    -   Part2/node2vec_linkedin.model
